# Gaga

## 项目介绍

本项目是本人在大一时学习Java利用课余时间写的一个dnf小游戏。

在原生Java swing的基础上实现任务动画，地图切换、任务移动、碰撞体积判定、背景音乐控制，欢迎试玩哈哈哈哈哈。

技术栈：JAVA Swing。
